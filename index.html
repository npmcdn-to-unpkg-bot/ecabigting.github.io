Project Description:

The TAG System is a combination of a web application and a command line interface. Its primary purpose is for supervisors/team leaders to keep track of team members time in and timeout as well as break in and break out.

The TAG systems main data is gathered from all of the door access devices that saves the time/in transaction from around the building. The data is then compared to users attendance and filed leave of absence. As well as the main production software that keeps track of team members productivity.

The CLI part of the system is run in a server that reads all the data in all the devices in the whole company building and then process it to fit the TAG systems main database. The ASP.net interface is for viewing, editing, and analysis of the data.

Project Description:

The Data Entry System - eDocs, is a data tracking software that allows companies to enter data on custom forms and attached huge image documents. The idea of this is that, a client company can request custom forms, that would allow them to save data via windows form and then be searchable and categorize afterwards. Another functionality of the system is that it allows scan documents to be attached to each data form created.

The system has two main parts, a windows form created from VB 6.0 which is used mainly in the production for data entry and document scanning, and a web interface created using asp.net forms. The web interface provides reporting capabilities as well as reports for current and up coming production load.
